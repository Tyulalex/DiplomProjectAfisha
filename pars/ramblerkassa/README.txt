  Модуль scecial_function.py содержит функции применимы только к конкретным событиям.
В модуле предусмотрено разделение прерывистями линиями с описанием применимости
функций к тому или иному событию.
  Аргументом функции является html после BeautifulSoup.

  При вызове функции scecial_function.fetch_film_info(bs) будет спарсена информация
о фильме, ны выходе будет словарь с ключами:
           'title ru' - название на русском,
           'title en' - название на английском,
           'genre' - жанр фильма,
           'age limit' - возрастное ограничение,
           'film description',
           'film len' - длина фильма,
           'film countrymaker' - страна - изготовитель фильма,
           'film producer' - режиссёр фильма,
           'film actors' - актёры.

  Вызов функции scecial_function.fetch_cinema_info(bs) запустит парсинг информации
по конкретному фильму в конкретном кинотеатре. На выходе словарь с ключами:
            'name' - название кинотеатра
            'link' - ссылка на кинотеатр
            'adress' - адрес кинотеатра
            'underground station' - станция метро, если такая имеется
            'sessions_info_2D' - информация по сеансам в 2D: минимальная и максимальная цена, время сеансов
            'sessions_info_3D' - информация по сеансам в 3D: минимальная и максимальная цена, время сеансов


  Функция scecial_function.fetch_events_date_and_place_info(bs) применяется для театров и концертов.
Функция парсит информацию о дате и месте проведения события. На выходе словарь с ключами:
            'place' - название места,
            'place url' - ссылка на место,
            'place adress' - адрес места,
            'underground station' - станция метро,
            'event date' - дата события


  Функция scecial_function.fetch_perfomance_info(bs) применяется для парсинга информации о спектакле.
На выходе словарь с ключами:
            'title' - название спектакля,
            'production' - постановка,
            'actors' - актёры,
            'producer' - режжисёр ,
            'price' - цена


  Функция scecial_function.fetch_concert_info(bs) парсит информацию о концерте.
На выходе словарь со значениями:
            'title' - название,
            'description' - описание,
            'price' - цена


===============================================================================
  Модуль general_function содержит функции общие для всех событий.


  Функция general_function.get_url_params(event) возвращает параметры для составления url API.
Результатом являются значения - event_city в виде чилса, currenttab в виде названия события.


  Полученные params в функции general_function.fetch_json(skip, event) подставляются в url API и открывают доступ к API.
Далее из API собираются все url выбранного события. Параметр skip служит для пролистывания url.


  Url записываются в файл в функции general_function.get_events_url_list(event).
Имя файла имеет вид '%s.txt' %event.


  Парсинг информации происходит при вызоые функции general_function.get_events_info(event).
В зависимости от выбранного события автоматически выбирается нужный файл со списком url, полученном
в предидущей функции, а так же применяеются подходящие функции из модуля special_function.


================================================================================
                                  Запуск.
================================================================================
  Создаётся файл с импортом BeautifulSoup, requests, json и модуля general_function.
Для получения файла со списком url вызывается команда:

general_function.get_events_url_list(event)

Обновление файла проводить раз в день или чаще.
Далее для получения информации о событиях вызывается функция:

general_function.get_events_info(event).
